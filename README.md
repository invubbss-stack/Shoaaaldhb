# ركن الجليد للعصائر والكنافة

واجهة شاشة منيو رقمية فاخرة لمحل **ركن الجليد للعصائر والكنافة — Al-Jaleed Corner for Juices & Kunafa**. التصميم مخصص للعرض على شاشة تلفزيون داخل المتجر بطابع طبيعي مشرق: رخام أبيض، خشب دافئ، فواكه طازجة، زجاجية ناعمة وحركة سينمائية باستخدام GSAP.

## هيكل المشروع

```text
index.html              # الواجهة، الأنيميشن، ومنطق القراءة من JSON
data/categories.json    # التصنيفات، ألوانها وأجواء الخلفيات
 data/products.json      # كل الأصناف والأسعار والسعرات والأحجام
 data/offers.json        # العروض المتحركة أعلى الشاشة
 data/settings.json      # بيانات المتجر، واتساب، العنوان، QR، العملة
sw.js                   # دعم مبدئي للأوفلاين عبر Service Worker
```

## تعديل المنيو مستقبلاً

لا تحتاج لتعديل الكود لإضافة سعر أو صنف. عدّل الملفات التالية فقط:

- `data/products.json` لتعديل المنتجات، السعرات، الصور، الأحجام والأسعار.
- `data/categories.json` لتعديل التصنيفات والألوان وأجواء الخلفية.
- `data/offers.json` لتعديل الشريط الترويجي المتحرك.
- `data/settings.json` لتعديل الهاتف، واتساب، العنوان، العملة ورابط QR.

## التشغيل محلياً

لأن الصفحة تقرأ ملفات JSON عبر `fetch`، شغّلها من خادم محلي:

```bash
python3 -m http.server 8000
```

ثم افتح:

```text
http://localhost:8000
```

## جاهزية مستقبلية

الهيكل الحالي مناسب للربط لاحقاً مع Firebase أو Supabase أو Node.js أو Laravel أو لوحة تحكم Admin Dashboard لأن بيانات المنيو مفصولة بالكامل في JSON وقابلة للتحويل لمصدر بيانات ديناميكي.
