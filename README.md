# ركن الجليد للعصائر والكنافة

واجهة منيو عربية تفاعلية لعرض منتجات العصائر والكنافة والحلويات مع شرائح مميزة، بطاقات منتجات، ساعة مباشرة، وعروض متحركة.

## التشغيل محلياً

افتح ملف `index.html` مباشرة في المتصفح، أو شغّل خادم ملفات ثابت مثل:

```bash
python3 -m http.server 8000
```

ثم افتح:

```text
http://localhost:8000
```

> تعتمد الصفحة على Tailwind CSS وخط Noto Sans Arabic وصور خارجية عبر CDN، لذلك تحتاج اتصال إنترنت لعرض التصميم والخطوط والصور بشكل كامل.
